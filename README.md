The Ebookstore is a lightweight e-commerce platform that mimics a real-world online bookstore. It connects a clean, responsive frontend to a simple backend server to manage inventory and process orders. The standout feature is the integration of a dummy PayPal gateway, allowing users to experience a realistic "sandbox" payment flow without using real currency.

**✨ Features**
Product Catalog: Dynamic display of books with titles, authors, and pricing.

Shopping Cart: Add, remove, and update book quantities in real-time.

Search & Filter: Easily find specific genres or titles using a search bar.

Dummy PayPal Integration: A simulated checkout process using the PayPal Sandbox API/Buttons.

Backend API: A simple server to handle book data and order confirmation.

Responsive Design: Fully optimized for a seamless experience on mobile and desktop.

**🛠️ Tech Stack**
Frontend
HTML5 & CSS3: For a professional, responsive user interface.

JavaScript (Vanilla): For cart logic, DOM updates, and API calls.

PayPal SDK: For implementing the dummy payment buttons.

Backend
Node.js & Express: To serve the frontend and manage the book database.
